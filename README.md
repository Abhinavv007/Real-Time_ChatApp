<h3>Welcome to our Real-Time Chat App! This application is built with Node.js and Socket.IO, providing a platform for users to engage in live conversations, exchanging messages instantly.</h3>

## Features

- **Real-Time Messaging:** Experience seamless and instantaneous communication with other users.
- **Emoji Support:** Express yourself with a wide range of emojis to enhance your messages.
- **Clean and Intuitive Design:** The app features a clean and intuitive design, ensuring that users can navigate seamlessly and focus on the conversations.
- **Multiple Rooms:** Join or create different chat rooms based on topics or interests.
